Discovery

Discovery is to find devices or network element or host from the network.

On this page you can Start a Discovery and scheduled it.

Discovery Type
PING : This Type of Discovery uses ICMP Protocol and finds the device or network element on Network
SNMP : This Type of Discovery uses SNMP Protocol and finds the device or network element on Network
UPNP : This Type of Discovery uses UPNP Protocol and finds the device or network element on Network
