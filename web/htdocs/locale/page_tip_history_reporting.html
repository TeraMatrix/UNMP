Backup & Restore

Here you can create backup , restore old backup or clean data of a particular month from system.

Backup Data: The system will create a backup of the data of selected month.

Restore Data: The system will restore a backup of the data of selected month. Reports of the selected month will be available after the data is restored .

Clean Data: This will clean the data of the selected month from the system. Please note that this method is irreversible.
