This page generates report for different Host Groups, Device types and different hosts.

On this page you can view report for different date and time periods along with different Host GroD types and different hosts .

Delta stands for the change in the consecutive values of fields .

Host Group: This drop down selects network Device Host Group for the report.

Device Type: This drop down selects Device type for the report.

Report Type:This drop down selects report type for the selected Device type.

Host Name:This drop down selects Hosts for the selected Device type . You can also search hosts here.

Date-Time Range:Here you select the date-time range for the report to be generated.

Actions
