Host

Host is a Network Element (Device). User can Manage and Monitor NE.

Active Host User can monitor these host.
Disable Host User cannot monitor these host but available in network.
Discovered Host Consists of Discovered hosts, The hosts which were discovered through TCP server or Manual discovery such as [PING,SNMP,UPNP].
Deleted Host contains deleted host, deleted host can be added back manually or by discovery.

On this page user can Add new host(s), Update host details and Delete host(s).

Add New Host
Update Host Details
Delete Host

Note: User can't delete or edit Local-host.
