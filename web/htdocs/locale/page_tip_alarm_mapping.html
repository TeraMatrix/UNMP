Alarm Masking

Alarm Masking is used co-relate the device alarms, where a user can categorize the device alarms as Current & Clear

Current Alarms: Alarms that are important to a user.
Clear Alarms: Alarms which means that a Specific Fault (Current Alarm) has been recovered.

Exception for Alarm Categorization: If the Device Vendor has marked some Device Event as an Alarm, that alarm can not be further categorized. UNMP System automatically categorizes such alarms.
