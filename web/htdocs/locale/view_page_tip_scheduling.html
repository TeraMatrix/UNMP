Scheduler

Planned repetitive device operations can be managed by the UNMP scheduler

List of Device Operations supported:

  • 1. Firmware Upgrade

  • 2. Radio Off/On: for Radio Devices Like AP, UBR

How to Use

  • Scheduler has been divided in two parts: Scheduling Screen [Displayed as calendar] & Scheduling Status

  • Scheduling an operation:

    • 1. On the Calendar displayed on the screen, click on desired Date & Time, "Create Event" pop-up will be displayed. Click on the "Create Event" link

    • 2. Select the Device Type which event has to be created

    • 3. Choose an operation from Event

    • 4. Adjust the Date & Time and the Interval of the Scheduling from Range

    • 5. If the Task has to be performed on regular basis, mark the task as Repetitive & Select the frequency of the operation to be performed

    • 6. Lastly, choose the Network Elements from Hosts for which the Scheduled Operations should run.

  • Edit a previously Scheduled Operation:

    • Already Scheduled Device Operations are visible on the Calendar, click on the Scheduled Operation that needs to be modified. A pop-up will appear on the screen with options:

      • # View Devices: To view the Hosts for which operation will be performed

      • # Edit: To edit the Operation summary

      • # Delete: To delete the Operation summary

    • Editing a Scheduled Operation is same as adding a New, explained in Scheduling an operation
