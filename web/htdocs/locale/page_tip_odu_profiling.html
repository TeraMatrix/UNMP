UBR Profile

RM18/RM Profiling of (Device). You can edit the profiling of individual device.

Search ProfileAt the top there is IP Address,MAC Address, and Device Type.By All Of that you can select an individual profile.If more than one device is come in search result then you can move to UBR Listing page

Radio Unit On this tab you can manage the channel bandwidth and country code.

UNMP On this tab you can manage the UNMP IP Address and Periodic Stats Timer.

UNMP Registration On this tab You can manage the address, contact person, mobile, alternate contact and email

Synchronization On this tab You can manage the Raster Time,Number of slaves,SyncLossThreshold,LeakyBucketTimer,SyncLossTimeOut,SyncTimerAdjust

ACL On this tab you can manage ACL Mode and MAC Addresses

Radio Frequency On this tab you can manage RF Frequency,RF Coding,MAC TX Power,Max CRC Errors,Leaky Bucket timer

Peer MAC On this tab we can manage the Number of slaves and Mac Addresses.

Commit To Flash This is a button on the bottom of Page.It save all your page data on device permanently

Reboot This is a button on the bottom of Page.It Reboots the device.When you press reboot there is a loading spin on you page and it stops spinning when device is reachable again after reboot.If device is not reachable after 100 sec then loading automatically hides and show you the message

Reconciliation This is a button on the bottom of Page.It save all device data on data storage and show on your page

Site Survey Result This is a button on the bottom of Page.It gives a pop up after runs successfully.It takes time to run.

BW Calculator This is a button on the bottom of Page.It gives a pop up after click on button.In this pop window there is TX Rate,TX Time and TX BW.TX Rate,TX Time are entered by you and bandwidth is calculated according to that

Form Working Description After Click On Save Button Of form.All The values given in form are going to set on the device.If All Values are set then there is a image is display after every Field.And OK button is display instead of save.After click on OK.the form is display with updated values. If values are not set then the image is display after fields which are not set.if No one field are set then it display after Every field on form . By clicking on retry image after fields ,the value of that field is again going to retry.When retry image is displayed then there are two buttons are also displayed Retry and Cancel button.When Click on retry button all fields are again going to set in which retry image is displayed.On click on cancel button the form is displayed with the updated values which are set and the retry values are discarded and the old values are displayed in that fields.
