Changing Personal information

To change the personal information

  1. Select Personal Information on tab menu
  2. Enter your details in the fields provided
  3. The omitted fields be left intact
  4. Click save to submit your personal information changes

Changing the Password

To change the password

  1. Select Change Password on tab menu
  2. Enter the old password
  3. Enter a new password. New password must be different from the old password and must follow the requirement of having
    • Minimum 2 alpha characters,
    • Minimum 2 numeric characters,
    • Minimum 2 special characters
    • Minimum length of password must be 8 characters
  4. Confirm your new password
  5. Click save to submit your password changes
