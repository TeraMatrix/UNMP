Analyzed Reports


Analyzed Reports differ from Simple Reports as Analyzed reports provide user with filtered data not just raw performance data.

How to Use

Host Group: This drop down selects Host Groups for the report.

Device Type: This drop down selects Device Type for the report.

Report Type:This drop down selects Report Type for the selected device Type.

Host Alias:This drop down selects Host for the selected device Type . You can also search hosts here.

Type:This drop down selects the Type of analyzed data.

Duration:This drop down selects the duration of analyzed data.

Time Range:Here you select the time range for the report to be generated.
Actions
: Call to action, to provide user with the calculated reports on the basis of parameters provided in the search form.
: Use for downloading the desired report in Excel format.
: Use for downloading the desired report in CSV format.
: Use to further filter the Report Parameters.
